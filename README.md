# Notepadd

Notepadd is a lightweight, Python-based text editor built with PySide6. Itâ€™s designed for simple note-taking while offering advanced features like tables, images, PDF export, and dark mode.

## Features
- Create new notes
- Save notes to any location (TXT or HTML)
- Export notes to PDF
- Print notes
- Adjust font and colors
- Dark/Light mode
- Multi-language support
- Insert tables and images

## Requirements
- Python 3.12+
- PySide6

## How to run
1. Open the dist folder.
2. Double-click `notepadd.exe` to run the app or use python > py notepadd.py
